# POV-Fan
A POV (Persistancy Of Vision) fan based on microchip PIC18F4520

## Environment
* MPLAB X IDE
* XC8 Compiler
* PicKit 3

## Hardware
* Green LED * 10
* Hall Effect Sensor
* Magnet
* IR Sensor
* IR Remote
* Lithium battery
* Regular Fan
